1 json是一种与开发语言无关的，轻量级的数据格式。全称是JavaScript Object Notation .

    
 2. 标准的json数据表示 。   (json中不含注释)
                    1.数据结构    Object  Array 
                    2. 基本类型     string  number   true   false   null        
           
           Object :  使用 {  }  花括号包含的键值对结构  ：其中   Key 必须是String 类型    value可以是任何基本类型或者数据结构  .
                                        key 和 value之间用冒号隔开  行尾用逗号 。
           Array  :   使用  [   ]   中括号来起始，并用逗号来分隔元素。
